(*** Helper: zero distance implies equality ***)
(*** LATEX VERSION: Axiom (1) of §20: d(x,y)=0 iff x=y. ***)
L75260
Let X, d, x and y be given.
L75261
Assume Hm: metric_on X d.
L75262
Assume Hx: x ∈ X.
L75263
Assume Hy: y ∈ X.
L75264
Assume H0: apply_fun d (x,y) = 0.
L75265
We prove the intermediate claim Hpos: ∀x0 y0 : set, x0 ∈ X → y0 ∈ X → ¬ (Rlt (apply_fun d (x0,y0)) 0) ∧ (apply_fun d (x0,y0) = 0 → x0 = y0).
L75268
Apply Hm to the current goal.
L75268
Assume Hcore Htri.
L75269
Apply Hcore to the current goal.
L75270
Assume Hcore2 Hposdef.
L75271
An exact proof term for the current goal is Hposdef.
L75272
We prove the intermediate claim Himp: apply_fun d (x,y) = 0 → x = y.
L75274
An exact proof term for the current goal is (andER (¬ (Rlt (apply_fun d (x,y)) 0)) (apply_fun d (x,y) = 0 → x = y) (Hpos x y Hx Hy)).
L75274
An exact proof term for the current goal is (Himp H0).
∎
