(*** LATEX VERSION: Homeomorphisms map open sets to open sets. ***)

L161892

L161893

L161896

L161896

L161898

L161899

L161900
Assume Hgpair: continuous_map Y Ty X Tx g ∧ image_of f U = preimage_of Y g U.
L161901
We prove the intermediate claim Hgcont: continuous_map Y Ty X Tx g.
L161903
An exact proof term for the current goal is (andEL (continuous_map Y Ty X Tx g) (image_of f U = preimage_of Y g U) Hgpair).
L161903
We prove the intermediate claim Heq: image_of f U = preimage_of Y g U.
L161905
An exact proof term for the current goal is (andER (continuous_map Y Ty X Tx g) (image_of f U = preimage_of Y g U) Hgpair).
L161905
rewrite the current goal using Heq (from left to right).
L161906
An exact proof term for the current goal is (continuous_map_preimage Y Ty X Tx g Hgcont U HUopen).
