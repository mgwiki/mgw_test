(*** helper: local finiteness of supports implies pointwise finiteness of nonzero values ***)
(*** LATEX VERSION: If only finitely many supports meet some neighborhood of x, then only finitely many functions are nonzero at x. ***)

Let X, Tx, P, x, N and F0 be given.
L190960

L190961

L190962

L190967

L190968

L190968

L190969

L190970

L190972

L190972

L190974

L190976

L190976
We prove the intermediate claim HxE: x ∈ Empty.
L190978

L190978

L190978
An exact proof term for the current goal is (EmptyE x HxE).
L190979
An exact proof term for the current goal is (Hctrl f HfP HSuppMeet).
∎
