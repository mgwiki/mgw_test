(*** helper: regular spaces carry a topology ***)
(*** LATEX VERSION: A regular space is in particular a topological space (used implicitly in the definitions). ***)
L128704
Let X and Tx be given.
L128705
Assume Hreg: regular_space X Tx.
L128706
We will prove topology_on X Tx.
L128707
We prove the intermediate claim Hone: one_point_sets_closed X Tx.
L128709
An exact proof term for the current goal is (andEL (one_point_sets_closed X Tx) (∀x : set, x ∈ X → ∀F : set, closed_in X Tx F → x ∉ F → ∃U V : set, U ∈ Tx ∧ V ∈ Tx ∧ x ∈ U ∧ F ⊆ V ∧ U ∩ V = Empty) Hreg).
L128713
An exact proof term for the current goal is (andEL (topology_on X Tx) (∀x : set, x ∈ X → closed_in X Tx {x}) Hone).
∎
