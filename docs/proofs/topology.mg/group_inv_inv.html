(*** helper: inverse of inverse in a group ***)
L260213
Let G, mult, inv, e and x be given.
L260214
Assume Hmultfun: function_on mult (setprod G G) G.
L260215
Assume Hinvfun: function_on inv G G.
L260216
Assume HeG: e ∈ G.
L260217
Assume Hassoc: ∀a b c : set, a ∈ G → b ∈ G → c ∈ G → apply_fun mult (apply_fun mult (a,b),c) = apply_fun mult (a,apply_fun mult (b,c)).
L260220
Assume Hid: ∀a : set, a ∈ G → apply_fun mult (e,a) = a ∧ apply_fun mult (a,e) = a.
L260222
Assume Hinv: ∀a : set, a ∈ G → apply_fun mult (a,apply_fun inv a) = e ∧ apply_fun mult (apply_fun inv a,a) = e.
L260225
Assume HxG: x ∈ G.
L260226
Set xi to be the term apply_fun inv x.
L260227
We prove the intermediate claim HxiG: xi ∈ G.
L260229
An exact proof term for the current goal is (Hinvfun x HxG).
L260229
Set xii to be the term apply_fun inv xi.
L260230
We prove the intermediate claim HxiiG: xii ∈ G.
L260232
An exact proof term for the current goal is (Hinvfun xi HxiG).
L260232
We prove the intermediate claim HxiiE: apply_fun mult (xii,e) = xii.
L260234
An exact proof term for the current goal is (andER (apply_fun mult (e,xii) = xii) (apply_fun mult (xii,e) = xii) (Hid xii HxiiG)).
L260234
We prove the intermediate claim Hexi: apply_fun mult (xi,x) = e.
L260236
An exact proof term for the current goal is (andER (apply_fun mult (x,apply_fun inv x) = e) (apply_fun mult (apply_fun inv x,x) = e) (Hinv x HxG)).
L260238
We prove the intermediate claim Hexii: apply_fun mult (xii,xi) = e.
L260240
An exact proof term for the current goal is (andER (apply_fun mult (xi,apply_fun inv xi) = e) (apply_fun mult (apply_fun inv xi,xi) = e) (Hinv xi HxiG)).
L260242
We prove the intermediate claim HeX: apply_fun mult (e,x) = x.
L260244
An exact proof term for the current goal is (andEL (apply_fun mult (e,x) = x) (apply_fun mult (x,e) = x) (Hid x HxG)).
L260244
rewrite the current goal using HxiiE (from right to left).
L260245
rewrite the current goal using Hexi (from right to left).
L260246
rewrite the current goal using (Hassoc xii xi x HxiiG HxiG HxG) (from right to left).
L260247
rewrite the current goal using Hexii (from left to right) at position 1.
L260248
rewrite the current goal using HeX (from left to right).
Use reflexivity.
∎
