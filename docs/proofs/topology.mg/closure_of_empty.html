(*** Helper: closure of empty set is empty ***)
(*** LATEX VERSION: The closure is the intersection of all closed sets containing A. ***)

Let X and Tx be given.
L45217

L45218
We will prove closure_of X Tx Empty = Empty.
L45219

L45221

L45221

L45222

L45222

L45223
Apply (SepE X (λx0 ⇒ ∀U : set, U ∈ Tx → x0 ∈ U → U ∩ Empty ≠ Empty) x Hx) to the current goal.
L45225
Assume HxX: x ∈ X.
(*** x :e closure means: x :e X and for all U open with x :e U, U ∩ Empty ≠ Empty; but U ∩ Empty = Empty, so this is impossible. ***)
L45226
Assume Hcond: ∀U : set, U ∈ Tx → x ∈ U → U ∩ Empty ≠ Empty.
L45227
We prove the intermediate claim HXopen: X ∈ Tx.
(*** Get X itself as an open set containing x ***)
L45230
An exact proof term for the current goal is (topology_has_X X Tx Htop).
L45230
We prove the intermediate claim HXne: X ∩ Empty ≠ Empty.
L45232
An exact proof term for the current goal is (Hcond X HXopen HxX).
L45232
We prove the intermediate claim HXempty: X ∩ Empty = Empty.
(*** But X ∩ Empty = Empty ***)
L45235
Apply set_ext to the current goal.
L45236
Let y be given.
L45236

L45236

L45239
An exact proof term for the current goal is HXempty.
L45241
An exact proof term for the current goal is (Subq_Empty (closure_of X Tx Empty)).
