(*** LATEX VERSION: A bijection f:X->Y is injective: if f(u)=f(v) then u=v. ***)
L101213
Let X, Y, f, u and v be given.
L101214
Assume Hbij: bijection X Y f.
L101215
Assume HuX: u ∈ X.
L101216
Assume HvX: v ∈ X.
L101217
Assume Heq: apply_fun f u = apply_fun f v.
L101218
Apply Hbij to the current goal.
L101219
Assume Hfun: function_on f X Y.
L101220
Assume Huniq.
L101221
We prove the intermediate claim HyY: apply_fun f u ∈ Y.
L101223
An exact proof term for the current goal is (Hfun u HuX).
L101223
Apply (Huniq (apply_fun f u) HyY) to the current goal.
L101224
Let x be given.
L101225
Assume Hx: x ∈ X ∧ apply_fun f x = apply_fun f u ∧ (∀x' : set, x' ∈ X → apply_fun f x' = apply_fun f u → x' = x).
L101227
We prove the intermediate claim Hxuniq: ∀x' : set, x' ∈ X → apply_fun f x' = apply_fun f u → x' = x.
L101229
An exact proof term for the current goal is (andER (x ∈ X ∧ apply_fun f x = apply_fun f u) (∀x' : set, x' ∈ X → apply_fun f x' = apply_fun f u → x' = x) Hx).
L101231
We prove the intermediate claim Hu: u = x.
L101233
Apply (Hxuniq u HuX) to the current goal.
Use reflexivity.
L101234
We prove the intermediate claim Hv: v = x.
L101236
Apply (Hxuniq v HvX) to the current goal.
Use symmetry.
L101237
An exact proof term for the current goal is Heq.
L101238
rewrite the current goal using Hu (from left to right).
L101239
rewrite the current goal using Hv (from left to right).
Use reflexivity.
∎
