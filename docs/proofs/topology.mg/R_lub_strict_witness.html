Let A, l and a be given.
Assume Hlub: R_lub A l.
Assume HAR: ∀x : set, x ∈ A → x ∈ R.
Assume HaR: a ∈ R.
Assume Halt: Rlt a l.
Apply (xm (∃x : set, x ∈ A ∧ Rlt a x)) to the current goal.
Assume Hex: ∃x : set, x ∈ A ∧ Rlt a x.
An exact proof term for the current goal is Hex.
Assume Hno: ¬ (∃x : set, x ∈ A ∧ Rlt a x).
Apply FalseE to the current goal.
We will prove False.
We prove the intermediate claim HubA: ∀x : set, x ∈ A → x ∈ R → Rle x a.
Let x be given.
Assume HxA: x ∈ A.
Assume HxR: x ∈ R.
We will prove Rle x a.
Apply (RleI x a HxR HaR) to the current goal.
Assume Haltx: Rlt a x.
Apply Hno to the current goal.
We use x to witness the existential quantifier.
Apply andI to the current goal.
An exact proof term for the current goal is HxA.
An exact proof term for the current goal is Haltx.
We prove the intermediate claim Hla: Rle l a.
An exact proof term for the current goal is (R_lub_min A l a Hlub HaR HubA).
We prove the intermediate claim Hnlt: ¬ (Rlt a l).
An exact proof term for the current goal is (RleE_nlt l a Hla).
An exact proof term for the current goal is (Hnlt Halt).
∎
