Let a and b be given.
Assume HaR: a ∈ R.
Assume HbR: b ∈ R.
Assume H0a: Rlt 0 a.
Assume Hb1: Rlt b 1.
Let x be given.
Assume Hx: x ∈ open_interval a b.
We will prove x ∈ unit_interval.
We prove the intermediate claim HxR: x ∈ R.
An exact proof term for the current goal is (SepE1 R (λx0 : set ⇒ Rlt a x0 ∧ Rlt x0 b) x Hx).
We prove the intermediate claim Hxprop: Rlt a x ∧ Rlt x b.
An exact proof term for the current goal is (SepE2 R (λx0 : set ⇒ Rlt a x0 ∧ Rlt x0 b) x Hx).
We prove the intermediate claim Hax: Rlt a x.
An exact proof term for the current goal is (andEL (Rlt a x) (Rlt x b) Hxprop).
We prove the intermediate claim Hxb: Rlt x b.
An exact proof term for the current goal is (andER (Rlt a x) (Rlt x b) Hxprop).
We prove the intermediate claim H0x: Rlt 0 x.
An exact proof term for the current goal is (Rlt_tra 0 a x H0a Hax).
We prove the intermediate claim Hx1: Rlt x 1.
An exact proof term for the current goal is (Rlt_tra x b 1 Hxb Hb1).
We prove the intermediate claim Hnltx0: ¬ (Rlt x 0).
Assume Hx0: Rlt x 0.
An exact proof term for the current goal is ((not_Rlt_sym 0 x H0x) Hx0).
We prove the intermediate claim Hnlt1x: ¬ (Rlt 1 x).
Assume H1x: Rlt 1 x.
An exact proof term for the current goal is ((not_Rlt_sym x 1 Hx1) H1x).
Apply (SepI R (λx0 : set ⇒ ¬ (Rlt x0 0) ∧ ¬ (Rlt 1 x0)) x HxR) to the current goal.
Apply andI to the current goal.
An exact proof term for the current goal is Hnltx0.
An exact proof term for the current goal is Hnlt1x.
∎
