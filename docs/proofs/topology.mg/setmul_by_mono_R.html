Let mult, U, V1 and V2 be given.
Assume Hsub: V1 ⊆ V2.
Let y be given.
Assume Hy: y ∈ setmul_by mult U V1.
We will prove y ∈ setmul_by mult U V2.
Apply (setmul_byE U V1 mult y Hy) to the current goal.
Let u be given.
Assume HuPack.
Apply HuPack to the current goal.
Assume HuU Hexv.
Apply Hexv to the current goal.
Let v be given.
Assume HvPack.
Apply HvPack to the current goal.
Assume Hv1 Hyeq.
rewrite the current goal using Hyeq (from left to right).
Apply (setmul_byI U V2 mult u v) to the current goal.
An exact proof term for the current goal is HuU.
An exact proof term for the current goal is (Hsub v Hv1).
∎
