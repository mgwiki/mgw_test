(*** helper: if an interval has empty intersection with K_set then no inv_nat lies in it ***)
(*** LATEX VERSION: If (a,b) cap K is empty, then no point of K (in particular no 1/n) lies in (a,b). ***)

Let a and b be given.
L20574

L20575
Let n be given.
L20576
Assume HnIn: n ∈ ω ∖ {0}.
L20576
Assume HinvIn: inv_nat n ∈ open_interval a b.
L20577
We will prove False.
L20578

L20580
An exact proof term for the current goal is (ReplI (ω ∖ {0}) (λm : set ⇒ inv_nat m) n HnIn).
L20580

L20582
An exact proof term for the current goal is (binintersectI (open_interval a b) K_set (inv_nat n) HinvIn HinvK).
L20582
We prove the intermediate claim HinvEmpty: inv_nat n ∈ Empty.
L20584
rewrite the current goal using Hcap (from right to left).
L20584

L20585
An exact proof term for the current goal is (EmptyE (inv_nat n) HinvEmpty).
