Let G, mult, inv, e, x and y be given.
Assume Hmultfun: function_on mult (setprod G G) G.
Assume Hinvfun: function_on inv G G.
Assume HeG: e ∈ G.
Assume Hassoc: ∀a b c : set, a ∈ G → b ∈ G → c ∈ G → apply_fun mult (apply_fun mult (a,b),c) = apply_fun mult (a,apply_fun mult (b,c)).
Assume Hid: ∀a : set, a ∈ G → apply_fun mult (e,a) = a ∧ apply_fun mult (a,e) = a.
Assume Hinv: ∀a : set, a ∈ G → apply_fun mult (a,apply_fun inv a) = e ∧ apply_fun mult (apply_fun inv a,a) = e.
Assume HxG: x ∈ G.
Assume HyG: y ∈ G.
Assume Hxy: apply_fun mult (apply_fun inv x,y) = e.
Set xi to be the term apply_fun inv x.
We prove the intermediate claim HxiG: xi ∈ G.
An exact proof term for the current goal is (Hinvfun x HxG).
We prove the intermediate claim Hxe: apply_fun mult (x,xi) = e.
An exact proof term for the current goal is (andEL (apply_fun mult (x,apply_fun inv x) = e) (apply_fun mult (apply_fun inv x,x) = e) (Hinv x HxG)).
We prove the intermediate claim HxE: apply_fun mult (x,e) = x.
An exact proof term for the current goal is (andER (apply_fun mult (e,x) = x) (apply_fun mult (x,e) = x) (Hid x HxG)).
We prove the intermediate claim HeY: apply_fun mult (e,y) = y.
An exact proof term for the current goal is (andEL (apply_fun mult (e,y) = y) (apply_fun mult (y,e) = y) (Hid y HyG)).
rewrite the current goal using HeY (from right to left).
rewrite the current goal using Hxe (from right to left) at position 1.
rewrite the current goal using (Hassoc x xi y HxG HxiG HyG) (from left to right).
rewrite the current goal using Hxy (from left to right).
rewrite the current goal using HxE (from left to right).
Use reflexivity.
∎
