(*** Helper: extract existence of inverse package from homeomorphism ***)
(*** LATEX VERSION: A homeomorphism has a continuous inverse. ***)
L68414
Let X, Tx, Y, Ty and f be given.
L68415
Assume H: homeomorphism X Tx Y Ty f.
L68416
An exact proof term for the current goal is (andER (continuous_map X Tx Y Ty f) (∃g : set, continuous_map Y Ty X Tx g ∧ (∀x : set, x ∈ X → apply_fun g (apply_fun f x) = x) ∧ (∀y : set, y ∈ Y → apply_fun f (apply_fun g y) = y)) H).
∎
