(*** helper: extract topology_on from locally_finite_family ***)
(*** LATEX VERSION: A locally finite family is defined only for a topological space (X,Tx). ***)
L196575
Let X, Tx and F be given.
L196576
Assume HLF: locally_finite_family X Tx F.
L196577
An exact proof term for the current goal is (andEL (topology_on X Tx) (∀x : set, x ∈ X → ∃N : set, N ∈ Tx ∧ x ∈ N ∧ ∃S : set, finite S ∧ S ⊆ F ∧ ∀A : set, A ∈ F → A ∩ N ≠ Empty → A ∈ S) HLF).
∎
