Assume Hg: function_on g A Y.
We will prove function_on (pair_map A f g) A (setprod X Y) ∧ ∀a : set, a ∈ A → ∃y : set, y ∈ setprod X Y ∧ (a,y) ∈ pair_map A f g.
Apply andI to the current goal.
We prove the intermediate claim HpmFS: pair_map A f g ∈ function_space A (setprod X Y).
An exact proof term for the current goal is (pair_map_in_function_space A X Y f g Hf Hg).
An exact proof term for the current goal is (function_on_of_function_space (pair_map A f g) A (setprod X Y) HpmFS).
Let a be given.
Assume Ha: a ∈ A.
We use (apply_fun f a,apply_fun g a) to witness the existential quantifier.
Apply andI to the current goal.
