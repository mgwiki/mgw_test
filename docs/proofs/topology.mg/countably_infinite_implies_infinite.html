Let X be given.
Assume Heq: countably_infinite X.
We will prove infinite X.
We prove the intermediate claim Heq': equip ω X.
An exact proof term for the current goal is (equip_sym X ω Heq).
We prove the intermediate claim Hinj: atleastp ω X.
An exact proof term for the current goal is (equip_atleastp ω X Heq').
An exact proof term for the current goal is (atleastp_omega_infinite X Hinj).
∎
