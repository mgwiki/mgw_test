(*** helper: upper bound for two elements in a directed set ***)
(*** LATEX VERSION: In a directed set J, for i,j in J there exists k in J with i <= k and j <= k. ***)

L106221

L106222
Assume HiJ: i ∈ J.
L106223
Assume HjJ: j ∈ J.
