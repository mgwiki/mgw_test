Let mult, U1, U2 and V be given.
Assume Hsub: U1 ⊆ U2.
Let y be given.
Assume Hy: y ∈ setmul_by mult U1 V.
We will prove y ∈ setmul_by mult U2 V.
Apply (setmul_byE U1 V mult y Hy) to the current goal.
Let u be given.
Assume HuPack.
Apply HuPack to the current goal.
Assume Hu1 Hexv.
Apply Hexv to the current goal.
Let v be given.
Assume HvPack.
Apply HvPack to the current goal.
Assume HvV Hyeq.
rewrite the current goal using Hyeq (from left to right).
Apply (setmul_byI U2 V mult u v) to the current goal.
An exact proof term for the current goal is (Hsub u Hu1).
An exact proof term for the current goal is HvV.
∎
