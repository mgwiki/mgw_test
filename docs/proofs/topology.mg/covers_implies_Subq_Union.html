(*** helper: cover yields subset of union ***)
(*** LATEX VERSION: If a family Fam covers X, then X is contained in Union Fam. ***)
L95437
Let X and Fam be given.
L95438
Assume Hcov: covers X Fam.
L95439
We will prove X ⊆ ⋃ Fam.
L95440
Let x be given.
L95441
Assume HxX: x ∈ X.
L95441
We will prove x ∈ ⋃ Fam.
L95442
We prove the intermediate claim Hexu: ∃u : set, u ∈ Fam ∧ x ∈ u.
L95444
An exact proof term for the current goal is (Hcov x HxX).
L95444
Apply (exandE_i (λu : set ⇒ u ∈ Fam) (λu : set ⇒ x ∈ u) Hexu (x ∈ ⋃ Fam)) to the current goal.
L95445
Let u be given.
L95446
Assume HuFam: u ∈ Fam.
L95447
Assume Hxu: x ∈ u.
L95448
An exact proof term for the current goal is (UnionI Fam x u Hxu HuFam).
∎
