(*** helper: build membership in closed_interval from endpoint inequalities ***)
(*** LATEX VERSION: If a≤x≤b and a,b,x are real then x belongs to [a,b]. ***)
L154252
Let a, b and x be given.
L154253
Assume HaR: a ∈ R.
L154254

L154255
Assume HxR: x ∈ R.
L154256

L154257

L154258
We will prove x ∈ closed_interval a b.
L154259
We prove the intermediate claim HCI_def: closed_interval a b = {t ∈ R|¬ (Rlt t a) ∧ ¬ (Rlt b t)}.
Use reflexivity.
L154261
rewrite the current goal using HCI_def (from left to right).
L154262

L154265

L154266
An exact proof term for the current goal is (RleE_nlt x b Hxb).
