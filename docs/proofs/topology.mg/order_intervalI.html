(*** Helper: introduction rule for order_interval membership ***)
(*** LATEX VERSION: If x is in X with a<x and x<b then x lies in the order interval (a,b). ***)
L38969
Let X, a, b and x be given.
L38970
Assume HxX: x ∈ X.
L38971
Assume Hax: order_rel X a x.
L38972
Assume Hxb: order_rel X x b.
L38973
An exact proof term for the current goal is (SepI X (λx0 : set ⇒ order_rel X a x0 ∧ order_rel X x0 b) x HxX (andI (order_rel X a x) (order_rel X x b) Hax Hxb)).
∎
