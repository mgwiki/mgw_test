Let X, x, n0, F0, e0, n1, F1, e1, n, F and e be given.
Assume HxX: x ∈ X.
Assume Hn0O: n0 ∈ ω.
Assume Hn1O: n1 ∈ ω.
Assume HnO: n ∈ ω.
Assume HFfin: finite F.
Assume HF0subF: F0 ⊆ F.
Assume HF1subF: F1 ⊆ F.
Assume HF0ne: ∃f : set, f ∈ F0.
Assume HF1ne: ∃f : set, f ∈ F1.
Assume HvalUI0: ∀f : set, f ∈ F0 → apply_fun f x ∈ unit_interval.
Assume HvalUI1: ∀f : set, f ∈ F1 → apply_fun f x ∈ unit_interval.
Assume HzeroOut0: ∀f : set, f ∈ F → ¬ (f ∈ F0) → apply_fun f x = 0.
Assume HzeroOut1: ∀f : set, f ∈ F → ¬ (f ∈ F1) → apply_fun f x = 0.
Assume Hbij0: bijection n0 F0 e0.
Assume Hbij1: bijection n1 F1 e1.
Assume Hbij: bijection n F e.
We will prove nat_primrec 0 (λk acc : set ⇒ add_SNo acc (apply_fun (apply_fun e0 k) x)) n0 = nat_primrec 0 (λk acc : set ⇒ add_SNo acc (apply_fun (apply_fun e1 k) x)) n1.
We prove the intermediate claim Hsum0: nat_primrec 0 (λk acc : set ⇒ add_SNo acc (apply_fun (apply_fun e0 k) x)) n0 = nat_primrec 0 (λk acc : set ⇒ add_SNo acc (apply_fun (apply_fun e k) x)) n.
An exact proof term for the current goal is (nat_primrec_sum_bijection_zero_outside_unit_interval X x n0 F0 e0 n F e HxX Hn0O HFfin HF0subF HF0ne HvalUI0 HzeroOut0 Hbij0 HnO Hbij).
We prove the intermediate claim Hsum1: nat_primrec 0 (λk acc : set ⇒ add_SNo acc (apply_fun (apply_fun e1 k) x)) n1 = nat_primrec 0 (λk acc : set ⇒ add_SNo acc (apply_fun (apply_fun e k) x)) n.
An exact proof term for the current goal is (nat_primrec_sum_bijection_zero_outside_unit_interval X x n1 F1 e1 n F e HxX Hn1O HFfin HF1subF HF1ne HvalUI1 HzeroOut1 Hbij1 HnO Hbij).
rewrite the current goal using Hsum0 (from left to right).
rewrite the current goal using Hsum1 (from right to left).
Use reflexivity.
∎
