(*** LATEX VERSION: A set is closed if it is the complement of an open set. ***)

L6838

L6839

L6840
Assume Hex: ∃U ∈ T, C = X ∖ U.
L6841
An exact proof term for the current goal is (andI (topology_on X T) (C ⊆ X ∧ ∃U ∈ T, C = X ∖ U) HT (andI (C ⊆ X) (∃U ∈ T, C = X ∖ U) HCsub Hex)).
∎
