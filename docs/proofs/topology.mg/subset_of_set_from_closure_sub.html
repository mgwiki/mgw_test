(*** helper: if closure(A) is contained in B then A is contained in B ***)
(*** from §48 and general topology: A is always contained in its closure ***)
(*** LATEX VERSION: Since A ⊆ closure(A), if closure(A) ⊆ B then A ⊆ B. ***)
